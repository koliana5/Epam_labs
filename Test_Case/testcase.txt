Сайт аренды автомобилей https://titanprokat.ru/

Кейс #1. Заказать звонок на несуществующий номер
Шаги:
1. Заходим на главную страницу
2. Заполняем форму "Зазать за 60 секунд"
 2.1 Выбрать любой класс
 2.2 Выбрать любой автомобиль
 2.3 Дата выдачи текущий день
 2.5 Дата возврата текущий день+1
 2.6 Ввод номера 8 555 555 (неполный номер)
3 Нажимаем кнопку "Забронировать"
Ожидаемый результат: Оповещение о неверном формате номера


Кейс #2. Оплата нулевой суммой
Шаги:
1. Перейти на вкладку "Оплата онлайн" (подпункт меню "Условия")
2. Заполняем форму "Оплатить услуги"
 2.1 Заполнить Ф.И.О
 2.2 Заполнить коментарий
 2.3 Сумма платежа установить = 0
3 Нажимаем кнопку "Отправить"
Ожидаемый результат: Оповеoщение пользователя о нулевой сумме платежа


Кейс #3. Просмотреть авто по марке
Шаги:
1. Выбрать пунк меню Автопарк
2. Выбрать подпунк меню бренды
3. Выбрать подпунк из меню бренды "Chevrolet"
Ожидаемый результат: отображение в подборке только автомобили марки "Chevrolet".

Кейс #4. Возможность заказа услуги "Транспортный аутсорсинг"
1. Выбрать пунк меню Услуги
2. Выбрать подпунк меню Транспортный аутсорсинг
Ожидаемый результат: отображение в подборке автомобилей предоставляемых по услуге "Транспортный аутсорсинг"

Кейс #5. Добавление дополнительных услуг
1. Заходим на главную страницу
2. Выбрать первое авто из класса эконом.
3. Нажать кнопку "Арендовать"
4. Добавить доп услугу "Ограничение ответственности при ДТП"
Ожидаемый результат: изменение цены аренды авто за день на стоимость услуги,пользователь уведомлеяется об этом

Кейс #6. Рассчитать стоимость аренды
1. Заходим на главную страницу
2. Выбрать первое авто из класса эконом.
3. Нажать кнопку "Расчитать"
4. Добавить доп услугу "Ограничение ответственности при ДТП"
5. Добавить доп услугу "Снижение возраста и стажа"
6. Установить дату выдачи в текущий день
7. Установить дату возврата = текущие день плюс десять дней
8. Нажать кнопку рассчитать
Ожидаемый результат: отображение всех платежей и итоговой суммы

Кейс #7. Просмотреть договор аренды
1. Выбрать пунк меню Полезно
2. Выбрать подпунк меню "договор аренды авто"
Ожидаемый результат: переход на страницу содержащей договор для ознакомления

Кейс #8. Орендовать авто без указания данных пользователя
1. Заходим на главную страницу
2. Выбрать первое авто из класса эконом.
3. Нажать кнопку "Арендовать"
4. Не заполняя параметры нажимаем "Забронировать"
Ожидаемый результат: Пользователю не может орендовать без указания данных, и просят его это сделать.

Кейс #9.Возможность просмотра контактов фирмы
1. Заходим на главную страницу
2. Выбрать вкладку «Контакты».	
Ожидаемый результат: на странице отобразятся контакты, которые в полной мере соответствуют компании и являются безошибочными.

Кейс #10. Аренда авто на будующее > 12 месяцев
1. Заходим на главную страницу
2. Выбрать первое авто из класса эконом.
3. Нажать кнопку "Арендовать"
4. Заполнить форму аренды авто (возможны невалидные значения например все "1")
5. Указать дату выдачи равную текущей дате + 13 месяцев
6. Нажать кнопку забронировать
Ожидаемый результат: вывод пользователю сообшения о невозможности бронирования автомобиля за время болшее года
